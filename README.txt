
Proyecto: Examen Final Práctico – Aplicaciones Web
Estructura incluida:
  /src/input.css
  /public/index.html
  /public/script.js
  /public/output.css
  package.json

Instrucciones:
  - Si quieres compilar Tailwind (recomendado), instala dependencias:
      npm install
    y luego corre:
      npm run build:css
    o para desarrollo:
      npm run watch:css

  - Abre public/index.html en el navegador para ver el proyecto.
  - La app usa la API pública de CheapShark (https://www.cheapshark.com/api/1.0).
